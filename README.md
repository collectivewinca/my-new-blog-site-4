# My New Blog Site 4

Experimental blog site iteration for testing content, layout, and deployment workflow.

## Purpose

This repository is used to validate blog-site changes quickly before promoting patterns to more stable projects.

## Expected Experience

- Fast setup for local iteration
- Lightweight content publishing flow
- Repeatable deployment for preview testing

## Getting Started

```bash
yarn install
yarn dev
```

## Notes
